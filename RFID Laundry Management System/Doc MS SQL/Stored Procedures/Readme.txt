CREATING
	GO TO YOUR SCHEMA (E.X. hlms)
		GO TO "Programmability"
			SELECT "Stored Procedures"
				LEFT CLICK AND CHOOSE "New Stored Procedure"
					REMOVE ALL THE DEFAULT CODES
					PASTE ALL CODES IN THE TEXT FILE
					EXECUTE IT
				
			REFRESH "Stored Procedures"

MODIFYING
	GO TO YOUR SCHEMA (E.X. hlms)
		GO TO "Programmability"
			GO TO "Stored Procedures"	
				LEFT CLICK YOUR STORED PROCEDURE FILE AND CHOOSE "Modify"
				MODIFY IT
				EXECUTE IT
